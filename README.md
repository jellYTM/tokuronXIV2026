## 実装内容

### 1. SIRモデルの実装（指定課題）
感染症の流行動態を記述する数理モデルである **SIRモデル**（Susceptible-Infected-Recovered）をPythonで実装しました。微分方程式の解法には `scipy.integrate.odeint` を用い、連続的な感染状況のシミュレーションを行っています。

### 2. 最小二乗法によるパラメータフィッティング（指定課題）
愛知県における新型コロナウイルス（COVID-19）の第5波（2021年7月〜9月）の実際の新規感染者数データを使用し、モデルパラメータの最適化を行いました。
* **手法**: 最小二乗法（`scipy.optimize.minimize` を使用した誤差二乗和の最小化）
* **成果**: 実際の流行曲線に対し、誤差を最小化する基本感染率（$\beta$）および回復率（$\gamma$）を自動推定しました。

### 3. 感染拡大抑制AIエージェントの構築（自由課題）
深層強化学習（**Deep Recurrent Q-Network: DRQN**）を用いて、感染抑制と経済維持のトレードオフを自律的に最適化するAI（政府エージェント）を仮想環境上に実装しました。

* **複合モデルの構築**:
    * **心理・行動モデル**: 繰り返される自粛要請による「市民の自粛疲れ（Fatigue）」と、それに伴う実効コンプライアンスの低下を定式化。
    * **マクロ経済モデル**: 自粛要請による労働力減少がGDPへ与える影響を、Cobb-Douglas型生産関数を用いて簡易的にモデル化。
* **アーキテクチャの高度化**:
    * **RNN（LSTM）の導入**: エージェントのニューラルネットワークにLSTM層を追加。過去7日間の感染トレンドや疲労度の推移を記憶し、時系列的な文脈に基づいた政策決定をシミュレートしました。
* **シミュレーション成果**: 
    * 対策を行わない場合の「医療崩壊による経済ストップ」と、自粛の長期化による「慢性的な経済損失」を比較。
    * 医療崩壊を回避しつつ、市民の疲労度をコントロールしながら、長期的（365日）なGDPの損失総和を最小化する戦略の学習プロセスを検証しました。
